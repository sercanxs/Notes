Java Components
Java Virtual Machine: Bytecodeleri okur. Bu bytecodeleri bilgisayar okuyamaz dolayısıyla bytecodelerin yorumlayıcısı budur.
Can run Java programs.it doesnt physically exist.
JRE= Java Runtime Environment. JVM'yi içerir. Java programlarını çalıştırabilmek için buna gerek vardır. Örneğin JVM matematiği
anlayamayabilir. JRE bunların yorumlanmasını sağlar, bytecodelere çevirir. it physically exists.
JDK= Java Development Kit. JRE'yi içerir. Kod yazmak için buna gerek vardır. İçinde kod yazabileceğimiz gerekli araçları içerir.

How Java Works

Write our code in any text editor and save it as classname.java
Compile our java code: javac classname.java
.class file will be created
Run your java code: java classname
